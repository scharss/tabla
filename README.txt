A Pen created at CodePen.io. You can find this one at https://codepen.io/m_/pen/RazNZW.

 Fetches data via AJAX from Google Sheets API endpoint.  This can allow one to programatically generate HTML based on Google Sheets data so that users/clients can easily edit things.  

If you want to pull this off with one of your own Google Sheets, make sure to do the following: 
*I n the Google Sheet, go to File > "Publish to the Web..."
* Under the "Published content & settings" dropdown, click "Start publishing"
* Your sheet is now publishing.  Grab the long UID from the url and use it in your JS.
